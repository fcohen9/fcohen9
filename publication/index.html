As Many As Six

This publication is about the importance of reading for children. It included 3 distinct sections: An article, an interview, and a photo essay.

process

jump to full publication

For this project, I knew from the get go that I wanted to use a lot of color and produce something whimsical and childlike, while still maintaining nice grid structures, clean typography, and an exciting but clear system. I was inspired by 2xElliotâ€™s rebranding for Brass for Africa, which contains the whimsy that I was drawn to but also maintains a very clear, clean system.

These are some of my initial pages. I originally was using an abitrary color pallete, seen in the frist two images. I had intended for the large pink pull quote to be a place of rest in my publication, but it actually ended up being distracting. It was also confusing because it looked too similar to my section dividers. I also had very small margins and dense text, which made my publication hard to read.
In my revisions, I sourced my color pallete from the images I was using, integrated the pull quotes more effectively, and coverted my dense 2 column layout into a one column with very large margins. These changes made my publication look better, but also served to express my content in a more effective way
